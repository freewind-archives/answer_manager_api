Answer Manager API
=====================================

This project is providing restful API to manage answers:
 
1. Record each answer submit by user
2. Get answers by user or by problem
3. Delete an answer

technology stack
================

1. Scala: 2.11.0
1. Play: 2.2.3
1. Database: h2
